# Fabian Sitaka — Environmental Consulting Website

Professional website for Fabian Sitaka, a NEMA Registered environmental 
consultant based in Nairobi, Kenya.

## Pages
- **Home** — `index.html`
- **Services** — `services.html`
- **About** — `about.html`
- **Insights** — `insights.html`
- **Book Consultation** — `book_consultation.html`
- **Contact** — `contact.html`

## Tech Stack
- HTML / Tailwind CSS
- Vanilla JavaScript
- EmailJS (consultation booking notifications)
- Netlify (hosting)

## Deployment
Deployed automatically via Netlify on every push to `main`.

Live site: https://fabiansitaka.netlify.app

## Local Development
No build step needed. Open any `.html` file directly in a browser 
or use a local server:

npx serve .
